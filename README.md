# future-letters
Letters From Your Future Self is an interactive landing page that lets users explore inspirational messages from their future self and reflect on their personal journey.

The website includes interactive flashcards that reveal wisdom when flipped, a collection of future-themed messages, and a section where users can write a letter to their future self. The interface features smooth animations, a galaxy-themed background, and a dark/light theme toggle to enhance the visual experience.

This project is built using HTML, CSS, and JavaScript and focuses on creating a visually engaging and reflective user experience through modern UI design and interactive elements.
